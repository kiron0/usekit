## Usage

```tsx {1, 4}
import { useWordleGame } from "@/hooks/use-wordle-game"

export function DailyWord() {
  const { currentGuess, guesses, submitGuess, setCurrentGuess } =
    useWordleGame()

  return (
    <div>
      <input
        value={currentGuess}
        maxLength={5}
        onChange={(event) =>
          setCurrentGuess(event.target.value.toUpperCase().slice(0, 5))
        }
      />
      <button onClick={() => submitGuess()}>Submit</button>
      <pre>{JSON.stringify(guesses, null, 2)}</pre>
    </div>
  )
}
```

## API Reference

### `useWordleGame(options?)`

The hook returns the active guess, evaluated guesses, keyboard state, win/loss state, and helpers for typing, submitting, and resetting.

Useful return values:

| Name            | Type                                                 | Description                                   |
| --------------- | ---------------------------------------------------- | --------------------------------------------- |
| `currentGuess`  | `string`                                             | Current in-progress guess.                    |
| `guesses`       | `GuessResult[]`                                      | Submitted guesses with per-letter evaluation. |
| `keyboardState` | `Record<string, "correct" \| "present" \| "absent">` | Best-known state for each typed letter.       |
| `submitGuess`   | `(input?: string) => boolean`                        | Validates and submits a guess.                |
| `resetGame`     | `() => void`                                         | Starts a new round.                           |

## Notes

- Duplicate letters are scored correctly, which is the main place Wordle clones go wrong.
- The dictionary and answer can both be injected for daily puzzles, multiplayer rounds, or deterministic tests, and the exact injected answer is still accepted even if it is not in the visible guess list.
- This hook is a good base for streak tracking and shareable summary grids.
